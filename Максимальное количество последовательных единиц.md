### Условие
Дан двоичный массив `nums`, вернуть _максимальное количество последовательных_ `1` в массиве.

### Пример

Ввод: числа = [1,1,0,1,1,1]
Выход:  3

### Решение

```go
func findMaxConsecutiveOnes(nums []int) int {
    maxCounter, counter := 0, 0
    for _, v := range nums {
		if v == 1 {
			counter++
		} else {
			if counter > maxCounter {
				maxCounter = counter
			}
			counter = 0
		}
	}

	if counter > maxCounter {
		maxCounter = counter
	}

	return maxCounter
}
```
