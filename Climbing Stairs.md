### Условие

Вы поднимаетесь по лестнице. Это занимает `n` шагов, чтобы достичь вершины.

Каждый раз вы можете подняться на `1` или `2` шага. Сколькими способами можно подняться на вершину?
### Пример
**Ввод:** n = 2
**Выход:** 2

**Ввод:** n = 3
**Выход:** 3
### Решение

```go
// идея состоит в том, чтобы с помощью рекурсии сочтавить дерево
// всех возможных исходов и посчитать количество благоприятных.
// чтобы аптимизировать алгоритм используется кэш, хранящий
// посчитанные ранее варианты

var cache = make(map[int]int)

func climbStairs(n int) int {
	// начальные проверки (base case)
    if n < 0 { // неблагоприятный исход (невозможные комбинации)
        return 0
    } else if n == 0 { // благоприятный исход
        return 1
    } else if val, ok := cache[n]; ok { // если данный случай уже был
	    // посчитан
        return val
    }
    // идем по дереву
    res := climbStairs(n-1)
    res += climbStairs(n-2)
	// сохраняем результат для данного случая
    cache[n] = res
    // возвращаем результат
    return res
}
```
