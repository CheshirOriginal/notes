### Условие
Дан массив `nums` целых чисел, верните количество чисел с четным количеством цифр.

### Пример

Ввод: числа = [12,345,2,6,7896]
Выход: 2

### Решение

```go
func findNumbers(nums []int) int {
    counter, result := 0, 0  
	for _, v := range nums {  
	    for v > 0 {  
	       counter++  
	       v /= 10  
	    }  
	  
	    if counter%2 == 0 {  
	       result++  
	    }  
	    counter = 0  
	}  
	return result
}
```
