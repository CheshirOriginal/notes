
### Условие

В этой задаче вам нужно определить, находятся ли две строки на расстоянии одного редактирования друг от друга. Две строки считаются находящимися на расстоянии одного редактирования друг от друга, если одну строку можно преобразовать в другую, выполнив ровно одну из следующих операций:

1. **Вставить один символ** : добавить ровно один символ в любую позицию строки, `s`чтобы сделать её равной строке `t`. Например, `"ab"`→ `"adb"`(вставить `'d'`).

2. **Удалить один символ** : удалить ровно один символ из строки `s`, чтобы она стала равна строке `t`. Например, `"abc"`→ `"ac"`(удалить `'b'`).

3. **Заменить один символ** : заменить ровно один символ в строке `s`на другой, чтобы сделать её эквивалентной строке `t`. Например, `"abc"`→ `"adc"`(заменить `'b'`на `'d'`).
### Пример

**Ввод:** - `"ab"` and `"acb"` (insert `'c'`)
**Выход:** true

**Ввод:** `"ab"` and `"ab"`
**Выход:** false
### Решение

```c++
class Solution {
public:
    bool isOneEditDistance(string s, string t) {
        int sLength = s.size();
        int tLength = t.size();

        // Гарантируем, что s всегда является строкой меньшей или равной длины для согласованности
        // Это упрощает нашу логику, уменьшая количество обрабатываемых случаев
        if (sLength < tLength) {
            return isOneEditDistance(t, s);
        }

        // Если разница в длине больше 1, строки не могут быть разделены одним редактированием
        if (sLength - tLength > 1) {
            return false;
        }

        // Перебираем более короткую строку (t)
        for (int i = 0; i < tLength; ++i) {
            // Когда мы находим первый отличающийся символ
            if (s[i] != t[i]) {
                // Случай 1: Одинаковая длина — должна быть 1 операция
                // замены
                // Проверяем, идентичны ли оставшиеся подстроки после
                // позиции i
                if (sLength == tLength) {
                    return s.substr(i + 1) == t.substr(i + 1);
                }
                // Случай 2: s длиннее на 1 — должна быть операция
                // удаления из s  
                // Пропустить символ в позиции i в s и сравнить
                // оставшиеся части
                return s.substr(i + 1) == t.substr(i);
            }
        }

        // Все символы совпали при сравнении 
        // Строки отличаются на одно редактирование только в том случае,
        // если s имеет ровно один дополнительный символ в конце
        return sLength == tLength + 1;
    }
};
```
